# 2026-03-10 FIF_1D optimization

- Prompt target: `tests/automated/test_1D_FIF_hurst.py`
- Baseline benchmark method: `LS2010`
- Optimized benchmark method: `LS2010_spectral`

## Result

- Baseline mean absolute H error across all parameter combinations and scaling ranges: `0.2687247706422018`
- New mean absolute H error across all parameter combinations and scaling ranges: `0.04882407407407406`
- Net improvement: about `5.50x` lower mean absolute error

## What changed

- Added `LS2010_spectral` kernel mode in `FIF_1D`.
- For noncausal 1D runs, the new mode builds kernels from the exact periodic spectral transfer function instead of the LS2010 real-space kernel.
- For causal runs, the new mode falls back to the original `LS2010` kernel path so existing causal behavior stays intact.
- Updated `tests/automated/test_1D_FIF_hurst.py` to benchmark `LS2010_spectral`.
- Added kernel tests covering the spectral kernel and identity behavior for the `H=0` observable case.

## Observations

- The main LS2010 failure mode was severe overestimation in the `1-10` range for intermittent noncausal cases.
- The spectral-response kernel largely removed those catastrophic short-range failures.
- Some high-`C1` noncausal cases still show bias, especially for `alpha=1.5`, `C1=0.1`, `H=0.0` and for `alpha=2.0`, `C1=0.1`, `H=0.7`.
